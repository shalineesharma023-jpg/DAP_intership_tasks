Big Data, Predictive and Sentiment Analysis Project

Project Overview
This project shows how to analyze data, understand patterns, predict future values, and visualize insights using different tools.

Dataset
The project uses the Tips dataset for analysis and an Amazon Sales dataset for dashboard creation.
Tips dataset includes total_bill, tip, gender, smoker, day, time, and number of people.

Big Data Analysis
Loaded the dataset using PySpark.
Explored and understood the data.
Calculated total and average bill.
Grouped data by day and gender.
Filtered high value customers and sorted top bills.

Predictive Analysis
Observed the relationship between bill and tip.
Used past data patterns to estimate future tips.

Sentiment Analysis
Classified text into positive, negative, and neutral.
Example:
Food is amazing → Positive
Very bad service → Negative

Amazon Sales Analysis using Power BI
Created an interactive dashboard using Amazon sales data.
Visualized total sales, profit, and orders.
Analyzed sales by region, category, and time.
Identified top performing products and trends.

Tools Used
Python, PySpark, Power BI

How to Run
Install pyspark using pip install pyspark
Run the file using python analysis.py

Conclusion
This project helps understand data, predict trends, and create dashboards for business insights.
